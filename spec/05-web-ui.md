# 웹 UI 설계 (Web UI Design)

## 전체 레이아웃

```
┌─────────────────────────────────────────────────────────────────┐
│  [AI 라이프]                          [프로필] [설정] [로그아웃] │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│                   2D 맵 (메인 화면)                              │
│                   [캐릭터들 실시간 이동]                         │
│                   [호감도/기분 이모지 표시]                     │
│                   [관계도 연결선]                                │
│                                                                 │
│           🚶 유리 (😊)  💖  🚶 민수 (🙂)                         │
│             ↕                                                      │
│           🚶 하늘 (😢)  💢  🚶 준우 (😤)                         │
│                                                                 │
│  ┌──────────────────────────────────────────────────────────┐  │
│  │  채팅창 (다른 사용자와 대화)                              │  │
│  │  [A] 안녕! 오늘 뭐해?                                      │  │
│  │  [B] 카페 가려는데 같이 갈래?                              │  │
│  │  [메시지 입력........................]     [전송]          │  │
│  └──────────────────────────────────────────────────────────┘  │
│                                                                 │
├─────────────────────────────────────────────────────────────────┤
│  [홈]  [캐릭터 목록]  [관계도]  [내 정보]                      │
└─────────────────────────────────────────────────────────────────┘
```

---

## 1. 메인 맵

### 기능
- 실시간 캐릭터 위치 표시
- 호감도/기분 이모지로 감정 표시
- 관계도 좋은 캐릭터 사이 연결선 표시
- 마우스/터치로 아바타 이동

### 캐릭터 표시
```
🚶 캐릭터 이름 (감정 이모지)
   ↓
호감도 이모지: 💖 좋아함, 👍 보통, 💢 싫음
```

---

## 2. 캐릭터 상세보기 (탭 시)

### 캐릭터 정보 팝업

```
┌─────────────────────────────────────────────────┐
│  📸 캐릭터 일러스트                              │
│                                                 │
│  👤 유리 (24세, 여성)                          │
│                                                 │
│  🎭 성격: 조용하지만 생각이 깊음...             │
│  🌟 관심사: 독서, 음악, 커피                    │
│  🏠 고향: 서울 북부                             │
│                                                 │
│  😊 기분: 행복                                  │
│  🔋 에너지: 85%                                 │
│                                                 │
│  💖 호감도: 72 (좋아함)                        │
│                                                 │
│  [대화하기]  [프로필 더 보기]  [닫기]           │
└─────────────────────────────────────────────────┘
```

## 3. 대화 화면 (미연시 스타일)

### 대화 인터페이스

```
┌─────────────────────────────────────────────────────────────────┐
│                       [뒤로]                                      │
│                                                                 │
│  ╔═════════════════════════════════╗                              │
│  ║                                 ║                              │
│  ║   📸 캐릭터 일러스트 (동적)    ║                              │
│  ║   (감정에 맞게 표정 변화)       ║                              │
│  ║                                 ║                              │
│  ╚═════════════════════════════════╝                              │
│                                                                 │
│  ┌─────────────────────────────────────┐                         │
│  │ 유리: "안녕하세요! 오늘 날씨가 참   │                         │
│  │       좋네요. 뭘 도와드릴까요?"      │                         │
│  └─────────────────────────────────────┘                         │
│                                                                 │
│  ┌─── 선택지 ─────────────────────────────┐                       │
│  │ [1] 네는 뭐 하는 사람이야?            │                       │
│  │ [2] 이 도시가 어때요?                  │                       │
│  │ [3] 함께 밥 먹을래요?                  │                       │
│  │ [4] 텍스트 입력: ........................ ]                    │
│  └────────────────────────────────────────┘                       │
│                                                                 │
│  💖 호감도: 72  😊 기분: 행복                                      │
└─────────────────────────────────────────────────────────────────┘
```

### 감정에 따른 일러스트 변화

| 감정 | 일러스트 변화 |
|------|--------------|
| happy | 웃는 표정, 밝은 색상 |
| sad | 아래로 향한 눈, 어두운 톤 |
| angry | 눈썹 꺾임, 붉은 색조 |
| excited | 눈 반짝임, 화려한 효과 |
| neutral | 평범한 표정 |

## 5. 내 정보 화면

- 아바타 수정
- 캐릭터 이름 변경
- 성격/관심사 설정
- 채팅 히스토리
- 관계 히스토리

---

## 6. 반응형 디자인

### 모바일 (세로)
- 맵 전체 화면
- 하단에 메뉴 버튼 5개
- 대화 시 풀스크린 대화창

### 태블릿/데스크톱 (가로)
- 왼쪽: 맵 (70%)
- 오른쪽: 채팅창/정보 패널 (30%)
