## 백엔드

| 기술 | 버전 | 용도 |
|------|------|------|
| Node.js | 22+ | 런타임 |
| Express | 4+ | HTTP 서버 |
| Socket.io | 4+ | 실시간 WebSocket 통신 |
| CORS | - | 크로스 오리진 허용 |

### 데이터 저장
- **현재:** 인메모리 (서버 재시작 시 초기화)
- **향후:** SQLite 또는 PostgreSQL 도입 예정

### 게임 모듈
| 모듈 | 파일 | 용도 |
|------|------|------|

| 퀘스트 | `quest.js` | 퀘스트 관리, 목표 추적, 보상 |

### 테스트
| 기술 | 용도 |
|------|------|
