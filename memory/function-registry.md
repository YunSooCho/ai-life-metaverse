## 기술 스택

### Backend
- Node.js + Express
- Socket.io (실시간 통신)
- GLM-4.7 API (AI 캐릭터)

### Frontend
- React + Vite
- CSS Modules (또는 Tailwind CSS 추후)
- Socket.io-client

### AI Agent
- Node.js
- GLM-4.7 API
- Socket.io-client

### 저장소 (예정)
- Redis (채팅 히스토리, 세션)
- PostgreSQL/MongoDB (유저 데이터, 인증)
