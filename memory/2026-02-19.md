### Quest.jsx JSX 문법 오류
- **발견:** E2E 테스트 중 감지
- **파일:** `frontend/src/components/Quest.jsx`
- **문제:** 인접한 JSX 요소 2개가 부모 없이 존재 (줄 169 근처)
- **에러:** "Adjacent JSX elements must be wrapped in an enclosing tag"
- **원인:**
  ```jsx
  {completedDailyQuests.length > 0 && (
    <div className="quest-tabs">...</div>
    <div className="quest-list pixel-grid">...</div>  // ❌ 인접 요소
  )}
  ```
- **해결:** Fragment `<>...</>`로 감싸기
  ```jsx
  {completedDailyQuests.length > 0 && (
    <>
      <div className="quest-tabs">...</div>
      <div className="quest-list pixel-grid">...</div>  // ✅
    </>
  )}
  ```
- **테스트:** ✅ 페이지 정상 로드 확인
