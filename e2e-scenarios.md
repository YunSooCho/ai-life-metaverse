## 📋 시나리오 목록

### S01. 초기 로딩
- [ ] 페이지 접속 → 정상 로드 (빈 화면 X)
- [ ] 헤더 표시: "🧞 AI 라이프 POC"
- [ ] 상태바: 나(플레이어), 방(메인 광장), 캐릭터 수, 연결 상태 ✅
- [ ] 하단 조작 안내 텍스트 표시
- [ ] favicon 로드 (콘솔 404 없음)

### S02. GameCanvas (메인 캔버스)
- [ ] 타일맵 배경 렌더링 (초록 배경 + 격자)
- [ ] 건물 5개 표시 (상점, 카페, 도서관, 공원, 체육관) + 이름 표시
- [ ] 플레이어 캐릭터 표시 + "플레이어" 이름
- [ ] AI 캐릭터 표시 + "AI 유리" 이름 (❌ 현재 "undefined" → Issue #37)
- [ ] 호감도 하트(💗) + 수치 표시
- [ ] 미니맵 표시 (우상단)
- [ ] 스프라이트 렌더링 (❌ 현재 fallback → Issue #38)

### S03. 시간/날씨 HUD
- [ ] 좌상단 HUD 박스 표시
- [ ] 게임 시간 표시 (HH:MM 형식)
- [ ] 시간대 이모지 (🌅🌤️🌇🌙 등)
- [ ] 날씨 상태 표시 (CLEAR/CLOUDY/RAIN/SNOW)
- [ ] 시간 경과에 따른 오버레이 색상 변화 (밤 어두워짐)
- [ ] 비/눈 날씨 시 파티클 효과

### S04. 캐릭터 이동
- [ ] 방향키 (↑↓←→) 이동 동작
- [ ] WASD 키 이동 동작
- [ ] 캔버스 클릭 이동 (터치 포함)
- [ ] 건물 충돌 시 이동 차단
- [ ] 맵 경계 이동 차단
- [ ] 이동 시 미니맵 위치 업데이트

### S05. 채팅 시스템
- [ ] 텍스트 입력 → 입력창에 표시
- [ ] SEND 버튼 클릭 → 메시지 전송 + 입력창 클리어
- [ ] Enter 키 → 메시지 전송
- [ ] Shift+Enter → 줄바꿈 (전송 안 됨)
- [ ] 메시지 전송 후 캐릭터 위 채팅 말풍선 표시
- [ ] AI 유리 채팅 응답 (GLM-4.7 API 연동)
- [ ] AI 응답 말풍선 표시

### S06. 방 메뉴 (ROOMS)
- [ ] 🏠 방 버튼 클릭 → 모달 열기
- [ ] "🌐 ROOMS" 헤더 표시
- [ ] 현재 방 목록 (메인 광장 등)
- [ ] 방 인원 수 표시 (N 👤)
- [ ] NEW ROOM NAME 입력 + CREATE 버튼
- [ ] ✕ 버튼으로 닫기

### S07. 인벤토리 (INVENTORY)
- [ ] 🎒 인벤토리 버튼 클릭 → 모달 열기
- [ ] "🎒 INVENTORY" 헤더 표시
- [ ] TOTAL 아이템 수 표시
- [ ] REFRESH 버튼 동작
- [ ] 아이템 있을 때: 그리드 목록 (아이콘, 이름, 수량, 설명)
- [ ] 소비 아이템 USE 버튼
- [ ] 아이템 없을 때: "INVENTORY EMPTY"
- [ ] ✕ 버튼으로 닫기

### S08. 보상 센터 (REWARD CENTER)
- [ ] 🎁 보상 버튼 클릭 → 모달 열기
- [ ] "🎁 REWARD CENTER" 헤더 표시
- [ ] 보상 목록: FIRST LOGIN, DAILY BONUS, ACHIEVEMENT
- [ ] 각 보상: PTS, EXP, 아이템 배지 표시
- [ ] CLAIM 버튼 동작
- [ ] 이미 클레임한 보상: CLAIMED 표시
- [ ] ✕ 버튼으로 닫기

### S09. 퀘스트 로그 (QUEST LOG)
- [ ] 📋 퀘스트 버튼 클릭 → 모달 열기
- [ ] "📋 QUEST LOG" 헤더 표시
- [ ] ACTIVE 탭 + 활성 퀘스트 수
- [ ] 퀘스트 카드: 타입 배지 (MAIN/SIDE), 제목, 설명
- [ ] OBJECTIVES 리스트 (○/✓ 체크, 진행률)
- [ ] 진행바 + 퍼센트 표시
- [ ] REWARD 섹션 (PTS, EXP, 아이템)
- [ ] 100% 완료 시 CLAIM REWARD 버튼
- [ ] AVAILABLE 탭 + 수락 가능 퀘스트
- [ ] ACCEPT 버튼 동작
- [ ] ✕ 버튼으로 닫기

### S10. 캐릭터 인터랙션 메뉴
- [ ] AI 캐릭터 클릭 → 인터랙션 메뉴 열기
- [ ] 캐릭터 이름 헤더
- [ ] INSA / GIFT / FRIEND / FIGHT 버튼
- [ ] 버튼 클릭 → 인터랙션 실행 + 호감도 변화
- [ ] 메뉴 외부 클릭 → 닫기

### S11. 이벤트 로그 / 기록
- [ ] 📊 기록 버튼 클릭 → 이벤트 로그 열기/닫기
- [ ] H 키로 히스토리 토글

### S12. 토스트 알림
- [ ] 인터랙션/이벤트 발생 시 토스트 표시
- [ ] success/warning/info 타입별 스타일
- [ ] 자동 사라짐

### S13. NPC 자동 행동 (AI Agent)
- [ ] AI 유리가 시간대별 이동 (스케줄 시스템)
- [ ] 이동 중 부드러운 애니메이션
- [ ] 활동 대사 자동 출력

### S14. 픽셀 아트 스타일 (Phase 3)
- [ ] Press Start 2P 폰트 적용 (헤더, 버튼, 입력)
- [ ] 픽셀 보더/그림자 스타일
- [ ] 버튼 hover/active 효과 (돌출/눌림)
- [ ] 레트로 색상 팔레트

2. **버그 수정 시** → 해당 시나리오에 체크 항목 추가
3. **기능 삭제 시** → 시나리오에 ~~취소선~~ 표시
4. **하트비트 E2E 실행 시** → 이 파일의 모든 시나리오 수행
